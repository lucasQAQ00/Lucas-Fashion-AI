# 05 内容生成助手

## 项目目标

基于商品资料和品牌规范，辅助生成一致、可复核的女装营销内容，提升内容生产效率。

## 使用场景

- 生成商品标题、卖点、详情页文案和搭配建议。
- 生成短视频脚本、直播话术、社媒笔记和活动文案。
- 针对不同平台、受众和语气改写已有内容。

## 技术方案

- 使用模板管理输入字段、品牌语气、渠道限制和输出格式。
- 通过可替换的模型适配层调用大语言模型，密钥仅从环境变量读取。
- 使用结构化输出、敏感词规则和事实字段校验降低错误风险。
- 保存提示词版本、输入摘要和人工修改记录，用于持续评估。

## 后续开发步骤

1. 整理品牌规范、平台规则和常用内容模板。
2. 定义商品信息输入模型与结构化输出格式。
3. 完成单商品、多模板生成的 MVP。
4. 加入敏感词、夸大宣传、事实一致性和人工审核。
5. 建立质量评测集并支持批量生成与内容版本管理。
