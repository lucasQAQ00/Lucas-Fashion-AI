# 03 本地图片整理工具

## 项目目标

对女装商品图片进行安全、可追溯的批量整理，统一命名和目录结构，减少重复素材并提升检索效率。

## 使用场景

- 按商品款号、品类、颜色、季节和图片类型归档。
- 识别重复图、近似图、低清图和异常尺寸图片。
- 批量生成商品图清单、缩略图和处理报告。

## 技术方案

- 使用 pathlib 和 Pillow 读取文件、尺寸及基础元数据。
- 使用文件哈希识别完全重复图片，后续使用感知哈希或视觉向量查找近似图片。
- 所有移动、重命名和删除操作先生成预览清单，确认后执行，并保留操作日志。
- 将分类规则写入可编辑配置，避免把业务规则硬编码在程序中。

## 后续开发步骤

1. 定义目标目录结构、命名规则和支持的图片格式。
2. 实现只读扫描与图片清单导出。
3. 增加重复检测、分类建议和操作预览。
4. 实现可回滚的批量整理流程。
5. 补充图形界面、测试和大批量性能优化。
