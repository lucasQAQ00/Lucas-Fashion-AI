# Lucas Fashion AI

面向女装行业的 AI 自动化系统集合，覆盖选品、直播内容加工、图片资产整理、利润分析、内容生成和竞品趋势监控。

## 项目结构

| 模块 | 目标 |
| --- | --- |
| `01_1688选品助手` | 辅助发现、筛选和评估 1688 女装货源 |
| `02_直播切片AI系统` | 自动识别直播高价值片段并生成短视频素材 |
| `03_本地图片整理工具` | 对本地商品图片进行分类、去重和规范命名 |
| `04_ERP利润分析系统` | 汇总 ERP 经营数据并分析商品利润 |
| `05_内容生成助手` | 生成女装商品与营销内容草稿 |
| `06_竞品趋势监控` | 跟踪竞品动态和女装市场趋势 |

## 开发原则

- 各模块相互独立，先验证最小可用流程，再逐步抽取公共能力。
- 使用 Python 3.11+，配置和密钥通过环境变量管理，不提交真实业务数据。
- AI 输出必须保留人工复核环节；采集数据时遵守平台规则和相关法律法规。
- 原始数据、处理中间数据和生成结果分层保存，便于追溯和审计。

## 快速开始

```bash
python -m venv .venv
# Windows PowerShell
.venv\\Scripts\\Activate.ps1
python -m pip install --upgrade pip
```

当前阶段仅初始化项目架构。进入对应模块阅读其 `README.md`，再根据优先级创建模块级依赖、配置、测试和业务代码。

## 推荐开发顺序

1. 明确每个模块的输入、输出、数据来源和验收指标。
2. 优先选择一个高价值模块完成最小可用版本（MVP）。
3. 为已验证模块建立 `src/`、`tests/`、配置示例和命令行入口。
4. 抽取日志、配置、模型调用和数据存储等公共能力。
5. 补充自动化测试、数据合规检查、部署和监控。
