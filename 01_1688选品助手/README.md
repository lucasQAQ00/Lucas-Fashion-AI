# 01 1688选品助手

## 项目目标

建立面向女装采购的智能选品流程，对候选商品进行结构化整理、条件筛选和综合评分，降低人工浏览与比价成本。

## 使用场景

- 按价格带、品类、风格、季节和起订量筛选货源。
- 对供应商、商品信息、销量信号和评价风险进行汇总。
- 形成候选款清单，供采购人员复核和打样决策。

## 技术方案

- 使用 Python 完成数据导入、清洗、规则过滤和评分。
- 优先支持合规的表格导入或平台授权接口；网页自动化仅用于被允许的场景。
- 使用 pandas 处理结构化数据，后续可接入视觉模型识别款式、颜色和版型。
- 将评分规则、平台参数与密钥放入独立配置或环境变量。

## 后续开发步骤

1. 确定选品字段、评分维度和合格阈值。
2. 准备脱敏样例数据并实现 CSV/Excel 导入。
3. 开发清洗、去重、筛选和评分的最小流程。
4. 增加人工复核结果回写与评分校准。
5. 补充测试、日志、异常处理和结果导出。
