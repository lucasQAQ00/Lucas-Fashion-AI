# 02 直播切片AI系统

## 项目目标

自动定位女装直播中的高价值内容片段，生成适合复用和二次编辑的短视频素材，提高直播内容利用率。

## 使用场景

- 提取单款讲解、试穿展示、价格机制和互动高峰片段。
- 自动生成字幕、标题、封面文案和切片摘要。
- 按商品、主播、主题和发布时间整理视频素材。

## 技术方案

- 使用 FFmpeg 完成音视频抽取、裁剪、转码和字幕合成。
- 使用语音识别生成带时间戳文本，并结合规则或大模型识别片段边界。
- 后续融合画面变化、商品识别、音量和互动数据进行多模态评分。
- 任务元数据使用 JSON 或 SQLite 保存，输出文件与源视频分开管理。

## 后续开发步骤

1. 明确输入视频格式、目标平台规格和切片标准。
2. 实现单文件转码、语音识别和字幕输出流程。
3. 构建基于关键词与时间戳的片段提取 MVP。
4. 加入画面分析、质量评分和人工预览确认。
5. 完善批处理、失败重试、版权检查和发布衔接。
