# 06 竞品趋势监控

## 项目目标

持续整理女装竞品、商品和内容变化，发现价格、款式、营销与市场趋势，为运营决策提供依据。

## 使用场景

- 监控竞品上新、价格、促销、热销款和内容主题变化。
- 汇总颜色、面料、版型、风格和季节性趋势信号。
- 自动生成日报、周报和异常变化提醒。

## 技术方案

- 优先使用平台授权接口、公开数据或人工导出的合规数据源。
- 使用 Python 进行定时采集、字段标准化、快照存储和变化检测。
- 使用 SQLite 起步，数据量增长后再迁移到 PostgreSQL 等数据库。
- 后续结合文本与视觉向量完成商品聚类、相似款比较和趋势摘要。

## 后续开发步骤

1. 确定竞品范围、监控指标、数据来源和更新频率。
2. 设计商品快照、价格历史和内容记录的数据模型。
3. 实现单一合规数据源的采集与变化检测 MVP。
4. 增加趋势聚合、报告生成和异常提醒。
5. 完善限流、失败重试、数据质量、合规审查和监控告警。
